# 野球スコア入力

スマホでも使える静的Webアプリです。

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成
2. このフォルダのファイルをアップロード
   - `index.html`
   - `styles.css`
   - `app.js`
   - `base-data.js`
   - `manifest.webmanifest`
   - `service-worker.js`
3. GitHubのリポジトリで `Settings` → `Pages`
4. `Deploy from a branch` を選択
5. Branchを `main`、Folderを `/root` にして保存
6. 表示されたURLをスマホで開く

## スマホで使う

iPhoneならSafariでURLを開き、共有ボタンから「ホーム画面に追加」を選ぶとアプリのように開けます。
